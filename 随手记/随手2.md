dhcp配置（在交换机上，dhcp工作在二层）：
基于接口的配置
```bash
dhcp enble
int g0/0/1.10 #以这个接口的地址作为地址池中的资源，并且网关就是它
dhcp select interface
dhcp server dns-list 114.114.114.114
dis this
```

基于地址池的配置
```
ip pool xxx
netbios-type
network 192.168.1.0 mask 24
gateway-list 192.168.1.1
dns-list 8.8.8.8
int g0/0/1.10
dhcp select global
```
